ПОЛУЧЕНИЕ ИНФОРМАЦИИ О ПОГОДЕ ИЗ Weather.com.

Данный пример кода иллюстрирует получение информации о погоде с сайта Weather.com.
Все работает через объекты XML, которые загружают с сайта XML код и остётся только взять из него нужную информацию.

Получение инфы идет в две стадии:
1. Нужно получить с сайта ID города, о котором будет инфа.
2. Загрузить опять таки с сайта уже полную инфу о погоде для данного города.
